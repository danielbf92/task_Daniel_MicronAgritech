--Create
--Read
--Update
--Delete

As well as the implementation of login with its encrypted password and validation.

## Setup
To run this project:

.env file must be created at the root of the project with the following data:

HOST=HOST
DATABASE=DATABASE
USER=USER
PASSWORD=PASSWORD

```
$ cd task_Daniel_MicronAgritech
$ npm run dev

```
